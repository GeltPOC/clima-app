# Clima App

Aplicación web para consultar el clima actual de cualquier ciudad del mundo.

## Características
- Búsqueda de ciudades con geocodificación Open-Meteo
- Temperatura actual, sensación térmica, mínima y máxima
- Condición meteorológica con iconos SVG animados
- Humedad relativa y velocidad del viento
- Diseño responsive y atractivo con Tailwind CSS
- Sin necesidad de API key

## Stack
- Next.js 16 (App Router)
- React 19
- TypeScript
- Tailwind CSS v3
- Open-Meteo API (geocodificación + clima)

## Desarrollo
```bash
npm install
npm run dev
```
